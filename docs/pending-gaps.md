# البنود المتبقية مقابل وثيقة المتطلبات (نظرة سريعة)

هذه قائمة مركزة بأهم البنود التي ما زالت ⚠️/❌ حسب وثيقة "طلبات-تطوير-برنامج-المحاسبة-والمخزون-كلاود". للتفاصيل الكاملة استخدم `docs/requirements-checklist.md`.

## تحسينات عامة
- شعار/صورة رئيسية جديدة، تعريب/ترجمة إنجليزية كاملة، أرصدة افتتاحية (حسابات + مخزون)، إقفال/إعادة فتح السنة، تنبيهات تلقائية موسعة وإعدادات ترقيم/طباعة/مخزون/حسابات/وضع POS باللمس.
- (منجز) تسلسل فواتير مستقل لكل مستخدم عبر إعداد per_user_sequence.
- وضع العمل دون اتصال: تمت إضافة حفظ فواتير POS/المبيعات محلياً مع مزامنة تلقائية عند عودة الاتصال (يحتاج تعميم لبقية الشاشات وتحسين المتابعة في الخلفية).

## لوحة المشتركين (SaaS)
- بناء لوحة دخول مستقلة بإدارة الشركات/الاشتراكات/المستخدمين والوثائق، مراقبة الاشتراكات (سارية/قريبة الانتهاء/منتهية)، تجديد سريع، وقراءة فقط لنسخة العميل.

## العملاء/الموردون/الأصناف/المندوبون
- تقارير عملاء/موردين (نشط/راكد/موقوف) بفلاتر تفصيلية.
- أصناف: شجرة تصنيفات غير محدودة، إدارة مستويات الأسعار الستة داخل بطاقة الصنف، خيار سعر شامل ضريبة، ضرائب متعددة (VAT + انتقائية)، إدارة تشغيلات متقدمة وتقاريرها، نوع الصنف (خدمة/مخزون) وخدمات مرتبطة ورسوم، هامش ربح تلقائي، تسعير حسب المستودع، ألوان/مقاسات/متغيرات، كرت صالون وتخصيص مستخدمين/أقسام، حجوزات بالموقع/الوقت، تسعير يتغير عند التحويل.
- مندوبون: بيانات موسعة، تنبيهات انتهاء وثائق، أرشفة، مستودع مخصص، عمولة ونسبة خصم، ربط مستوى سعر.

## حسابات ختامية وحسابات
- إعداد القوائم (الدخل/الإجمالي/المتاجرة/الأرباح والخسائر/الميزانية).
- شجرة حسابات متعددة المستويات مع مراكز تكلفة، قيود افتتاحية/يومية/إقفال، سندات قبض/صرف متعددة البنود، احتساب ضريبة المصروفات المتنوعة.

## مبيعات/مشتريات/عروض/تحويلات
- مبيعات: ربط محطات الشبكة المحددة، اكتمال سيناريوهات الدفع.
- مشتريات: دعم شامل للضريبة شامل/غير شامل، إدارة التشغيلات والبحث عنها، تمكين تعديل الفاتورة وبحث متقدم.
- عروض أسعار: CRUD كامل، تحويل لفاتورة مع اختيار الدفع، ضريبة شامل/غير شامل، تقرير بفلاتر.
- عروض ترويجية: عروض متعددة الأصناف بسعر عرض، باركود/تشغيلة، ربط مندوب وزمن/كمية، خصم تلقائي من المخزون، حالات التفعيل/إيقاف/إنهاء/حذف.
- تحويل مخزني: دورة طلب/موافقة/رفض مع أسباب وتقارير، وإرسال للتالف.

## التقارير
- حزمة تقارير المحاسبة (فلاتر شاملة، ضرائب إضافية، يومية، كشوف حساب، مطابقات، الأستاذ العام، أرصدة تفصيلية، قيود وسندات، حركة شهرية، فوق الحد).
- تقارير المخزون (كميات/أسعار/تكلفة/حركة/تحليل أعمار/الأكثر والأقل/الراكدة/مطابقة الجرد)، قوائم الملفات، إجماليات الحركة.
- تقارير العملاء والموردين (أرصدة، أعمار ديون، حركة، عملاء/موردون راكدون، حركات شهرية/حسب صنف/فرع).
- تقارير المناديب (مبيعات/تحصيلات، عمولات نسبية/ثابتة، أعمار تحصيلات).
- إبراز مردودات في التقارير وعدم جمعها ضمن الإجماليات، واستمرار زر المرتجع حتى اكتمال الاسترجاع.

## الجرد
- إظهار أرقام التشغيلات وكميات المجرد/المتبقي لكل تشغيلة، وخيارات تسوية المخزون تلقائياً حسب نتائج الجرد.

## نقاط البيع (POS)
- تحسين شاشة اللمس بالصور/المجموعات واختصارات POS، أوضاع مطاعم (محلي/سفري/طاولة/مناطق)، تعريف الشفتات، تقارير POS (نهاية يوم، كميات، شفت، مستخدم، نقدي/شبكة، أرباح).

## التصنيع
- شاشة تصنيع (BOM بسيط) لخصم المواد الخام وإضافة المنتج التام للمخزون مع ربط عبر API.
